--# Healthcare Pharmacy & Patient Analytics

## Project Goal
To analyze pharmacy revenue drivers and patient trends across Kenya

## Technical Process
* **Data Prep:** Cleaned and transformed raw Excel data using Power Query to handle nulls and standardize formats.
* **Modeling:** Built a Star Schema Model connecting Patients, Visits, Drugs, and Dates via a central Transaction Fact Table.
* **DAX:** Developed measures to track key KPIs: Total Revenue ($200k), Distinct Visits (92), and Average Length of Stay (1-2 days).
* **UI/UX:** Implemented a Page Navigator for seamless dashboard storytelling.
